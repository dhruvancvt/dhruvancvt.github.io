Motion Tracking

Implemented a movement tracker using the MPU6050 9-axis Inertial sensor on an Arduino. I transferred data using the I2C interface and then modified the accelerometer data using Python to generate velocity and displacement through successive integration. Further improved accuracy by writing a moving average low pass filter to reduce noise from the module.
