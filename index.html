Portable Executable (PE) Injection Using Python

This project showcases a proof-of-concept implementation of Portable Executable (PE) injection using Python and the ctypes library. The code demonstrates fundamental techniques involved in process injection: including memory allocation, relocation table parsing, and remote thread creation in a Windows environment. DISCLAIMER: This project is only for educational purposes only. It is not intended for malicious use and should be handled responsibly.
